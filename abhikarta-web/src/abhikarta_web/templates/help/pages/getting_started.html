System Requirements

Hardware
  • 2+ CPU cores
  • 4GB+ RAM (8GB recommended)
  • 10GB disk space
  • Optional: GPU for local LLMs
Software
  • Python 3.11+
  • pip 23.0+
  • Git 2.30+
  • Optional: PostgreSQL 14+
Browser
  • Chrome 90+
  • Firefox 88+
  • Edge 90+
  • Safari 14+

Installation

Step 1: Clone Repository
# Clone the repository
git clone https://github.com/your-repo/abhikarta-llm.git
cd abhikarta-llm
Step 2: Create Virtual Environment
# Create and activate virtual environment
python -m venv venv

# Linux/Mac
source venv/bin/activate

# Windows
venv\Scripts\activate
Step 3: Install Dependencies
# Install required packages
pip install -r requirements.txt

Configuration

Edit config/application.properties to configure your installation:

# Application Settings
app.name=Abhikarta-LLM
app.version=1.3.0

# Server
server.host=0.0.0.0
server.port=5000
server.debug=false
server.secret_key=change-this-to-a-random-secret-key

# Database (sqlite or postgresql)
database.type=sqlite
database.sqlite.path=./data/abhikarta.db

# LLM Providers (add your API keys)
llm.openai.api_key=sk-your-openai-key
llm.anthropic.api_key=sk-ant-your-anthropic-key
llm.default_provider=openai

Start the Server

# Start the server
python run_server.py
