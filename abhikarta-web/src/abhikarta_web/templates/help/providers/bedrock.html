AWS Bedrock

Multi-model access through Amazon Web Services

Overview

Amazon Bedrock provides serverless access to foundation models from multiple providers including Anthropic, Meta, Mistral, Cohere, and Amazon's own Titan models. Ideal for enterprises already in the AWS ecosystem.

Strengths

Multi-Model Access

Single API for Claude, Llama, Mistral, Titan, and more.

AWS Integration

Native integration with IAM, VPC, CloudWatch, and other AWS services.

Enterprise Security

Data never leaves your AWS account, no external API calls.

Guardrails

Built-in content filtering and safety guardrails.

Weaknesses

AWS Lock-in

Requires AWS account and infrastructure knowledge.

Model Lag

New models may not be immediately available on Bedrock.

Configuration

# config/application.properties
llm.bedrock.enabled=true
llm.bedrock.region=us-east-1
llm.bedrock.access_key_id=AKIAXXXXXXXXXXXXXXXX
llm.bedrock.secret_access_key=your-secret-key
llm.bedrock.default_model=anthropic.claude-sonnet-4-5-20250929-v1:0
Tip: Use IAM roles instead of access keys when running on EC2 or ECS.

Usage Examples

{
