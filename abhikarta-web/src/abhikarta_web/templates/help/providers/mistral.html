Configuration in Abhikarta-LLM

Step 1: Get Your API Key
  1. Go to console.mistral.ai
  2. Create an account
  3. Navigate to API Keys and create a key
Step 2: Configure in application.properties
# config/application.properties
llm.mistral.enabled=true
llm.mistral.api_key=your-mistral-api-key
# Or: llm.mistral.api_key=${MISTRAL_API_KEY}

llm.mistral.default_model=mistral-large-latest
llm.mistral.base_url=https://api.mistral.ai/v1
