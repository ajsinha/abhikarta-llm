What is MCP?

Model Context Protocol (MCP) is an open protocol that enables AI assistants to connect with external tools and data sources.

Connection Types
  • HTTP/SSE: Connect to a remote MCP server running on a URL
  • STDIO: Launch a local MCP server process
Example STDIO Commands
npx @modelcontextprotocol/server-filesystem /path
python mcp_server.py
node mcp-server.js
