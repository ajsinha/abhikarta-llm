## If You Have Permission to Contribute

If you have received written permission and signed the CLA:

### Code Style

- Follow PEP 8 guidelines
- Use type hints throughout
- Write comprehensive docstrings
- Maintain existing copyright headers
- Add unit tests for new features

### Commit Messages

Format:
```
[TYPE] Brief description

Detailed explanation of changes

Signed-off-by: Your Name <your.email@example.com>
```

Types: FEAT, FIX, DOCS, TEST, REFACTOR, STYLE, CHORE

### Testing

```bash
# Run tests
python -m pytest tests/

# With coverage
python -m pytest --cov=facades tests/

# Specific test
python -m pytest tests/test_facades.py::TestChatCompletion
```

### Documentation

- Update relevant README files
- Add examples for new features
- Document breaking changes
- Update CHANGELOG.md

### Code Review Process

1. Submit code for review
2. Address feedback
3. Obtain approval from copyright holder
4. Code will be merged by authorized personnel
