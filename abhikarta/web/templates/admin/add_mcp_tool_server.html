Expected API Response

The tools endpoint should return JSON:

{
  "tools": [
    {
      "name": "tool_name",
      "description": "What it does",
      "inputSchema": {
        "type": "object",
        "properties": {...}
      }
    }
  ]
}
Authentication Types

Bearer Token:

Token is sent in Authorization header as "Bearer {token}"

API Key:

Key is sent in custom header (e.g., X-API-Key)

Basic Auth:

Calls auth endpoint with credentials, uses returned token for subsequent calls

Auth Endpoint Response

For Basic Auth, the endpoint should return:

{
  "token": "jwt-token-here",
  "user": {
    "user_id": "123",
    "user_name": "admin",
    "roles": ["admin"]
  }
}

Or just the raw token string if not JSON.
