About MCP Tool Servers
What are MCP Tool Servers?

MCP (Model Context Protocol) Tool Servers are external services that expose tools via a REST API. These tools can be dynamically loaded into Abhikarta-LLM at runtime, allowing agents to use them without restarting the application.

Expected API Format

The tools endpoint should return a JSON response with a tools array, where each tool has name, description, and inputSchema.

Authentication Types
  • None - No authentication required
  • Bearer - Bearer token in Authorization header
  • API Key - Custom header with API key
  • Basic - Basic HTTP authentication
