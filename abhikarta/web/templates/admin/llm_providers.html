Key Terms on This Page
LLM (Large Language Model)

An AI model that understands and generates human-like text (e.g., GPT-4, Claude).

LLM Provider

A service hosting LLMs via APIs (OpenAI, Anthropic, Google, AWS, etc.).

API Key

A unique identifier for authenticating requests to an LLM provider.

Model

A specific version of an LLM (e.g., "gpt-4o", "claude-3-sonnet").

Temperature

Controls output randomness: lower = focused, higher = creative.

Inference

The process of using a trained model to generate text completions.
