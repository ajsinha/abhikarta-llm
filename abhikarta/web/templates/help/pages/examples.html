Running Examples

Prerequisites
# Install dependencies
pip install langchain-ollama

# Ensure Ollama is running
ollama serve

# Pull the model
ollama pull llama3.2:3b
Running Workflow Examples
# Navigate to examples directory
cd examples/workflow/

# Run document processing with sample data
python document_processing_pipeline.py --sample

# Run content generation interactively
python content_generation_pipeline.py --interactive

# Run strategic planning demo
python strategic_planning_pipeline.py --demo
Running Agent Examples
# Navigate to examples directory
cd examples/agent/

# Start research assistant
python research_assistant.py --interactive

# Review a code file
python code_assistant.py --review your_file.py

# Start tutoring session
python interactive_tutor.py --topic "machine learning"

# Quick decision analysis
python decision_support_system.py --quick "Should I accept the job offer?"
Customization: All examples support --model and --base-url flags to use different LLM providers or models.
