What is an Agent Swarm?
Event-Driven Architecture

Swarms use an internal event bus for agent communication. A master actor choreographs agent activities based on intelligent decisions.

On-Demand Scaling

Agents are spawned on-demand when events match their subscriptions. Only the master actor stays resident in memory.

External Triggers

Swarms can be triggered by Kafka messages, HTTP webhooks, scheduled events, or direct user queries.
