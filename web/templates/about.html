Platform Overview

Abhikarta is a comprehensive, enterprise-grade Large Language Model (LLM) platform designed to empower organizations with cutting-edge AI capabilities. Built on a robust, scalable architecture, Abhikarta provides seamless integration with multiple AI providers, sophisticated workflow orchestration, and advanced session management—all within a secure, production-ready environment.

Core Features

Multi-Provider Integration

Seamlessly integrate with 13+ leading AI providers including OpenAI, Anthropic, Google, Cohere, Mistral, Meta, Groq, HuggingFace, Together AI, Ollama, AWS Bedrock, and more. Switch between providers dynamically with zero downtime.

Workflow Orchestration

Design complex AI workflows using our visual workflow designer with support for 7 node types: Start, End, LLM, Tool, Human-in-the-Loop, Conditional, and Parallel execution. Export, import, and version control your workflows with ease.

Advanced Session Management

Industry-leading session-based facade caching delivers 90% performance improvements for subsequent requests. Automatic cleanup, 60-minute TTL, and intelligent cache invalidation ensure optimal resource utilization.

14 Execution Modes

Support for Chat, DAG, ReAct, Planning, Autonomous, Human-in-the-Loop, Background, RAG, Chain of Thought, Tool Calling, Multi-Agent, Hybrid, Kafka Listening, and custom execution modes. Choose the right paradigm for your use case.

Comprehensive Data Persistence

SQLite-based persistence layer with 19 tables managing sessions, conversations, workflows, executions, tool calls, and audit logs. Full conversation history with efficient retrieval and analysis capabilities.

Enterprise Security

Role-based access control, secure session management, comprehensive audit logging, and PII protection. All interactions logged for compliance and security analysis with configurable retention policies.

Real-Time Analytics

Monitor usage patterns, track token consumption, analyze cost metrics, and gain insights into model performance. Comprehensive dashboards for administrators and users with exportable reports.

MCP Integration

Model Context Protocol integration enables seamless tool calling and external service connectivity. Build custom tools, integrate APIs, and extend platform capabilities with standardized protocols.

Agent Orchestration

Advanced agent systems supporting autonomous operation, multi-agent collaboration, dynamic tool selection, and goal-oriented task completion. ReAct and Planning modes enable sophisticated reasoning workflows.

System Architecture

Layered Architecture with Clean Separation of Concerns

Presentation Layer

  • Flask Web Framework - RESTful API endpoints and web interface
  • Bootstrap 5 UI - Responsive, mobile-first design
  • Jinja2 Templates - Dynamic server-side rendering
  • JavaScript/AJAX - Real-time interactions and updates
  • D3.js Visualization - Workflow designer and analytics

Business Logic Layer

  • LLM Facades - Unified interface for 13+ providers
  • Session Manager - Thread-safe session lifecycle management
  • Workflow Engine - DAG execution with LangGraph integration
  • Execution Modes - 14 different execution paradigms
  • Caching Layer - Redis-based facade caching (90% speedup)

Integration Layer

  • LangChain Framework - Chain composition and tool integration
  • LangGraph - State machine and graph-based workflows
  • MCP Protocol - Model Context Protocol for tool calling
  • Kafka Integration - Event streaming and message processing
  • REST APIs - External service integration

Data Layer

  • SQLite Database - 19 tables for comprehensive data management
  • Session Store - User sessions and authentication state
  • Conversation History - Full context preservation and retrieval
  • Workflow Definitions - JSON-based workflow specifications
  • Audit Logs - Complete interaction history and compliance
