Bulk create provider and models from JSON file

Instructions
JSON Format Requirements:
  • File must be valid JSON format
  • Must contain provider, api_version fields
  • Must contain models array with at least one model
  • Each model must have name, version, and description
  • Optional fields: base_url, notes, model capabilities, costs, strengths
What happens on upload:
  1. File is validated for correct JSON structure
  2. Provider is created (or updated if exists)
  3. All models are created with their specifications
  4. Capabilities, strengths, and costs are populated
  5. Model registry is automatically reloaded
Important Notes:
  • If provider exists, it will be updated
  • Existing models will be skipped (not overwritten)
  • Invalid JSON will be rejected
  • All fields are validated before insertion
  • Maximum file size: 10MB
