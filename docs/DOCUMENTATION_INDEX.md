## 📥 Getting the Documentation

### Option 1: Download Package
```bash
# Download complete package (626 KB)
wget abhikarta-llm-v3.1.2-COMPLETE.tar.gz

# Extract
tar -xzf abhikarta-llm-v3.1.2-COMPLETE.tar.gz
cd abhikarta-llm

# All docs are included!
ls *.md
```

### Option 2: Read Online
All documentation is in markdown format, readable in any text editor or on GitHub.

### Option 3: Generate HTML
```bash
# Convert to HTML (requires markdown tool)
pip install markdown
python -m markdown README.md > README.html
```
