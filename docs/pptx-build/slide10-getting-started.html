Getting Started

Up and running in 5 minutes with Ollama (free, local).

1

Install Ollama

curl -fsSL https://ollama.com/install.sh | sh
2

Pull a Model

ollama pull llama3.3
3

Clone Abhikarta

git clone https://github.com/abhikarta/abhikarta-llm
4

Start the Server

python run.py
5

Open Dashboard

http://localhost:5000

Configuration

config/application.properties

# Server

server.host=0.0.0.0

server.port=5000

# Default Provider (Ollama)

llm.default_provider=ollama

llm.ollama.enabled=true

llm.ollama.base_url=http://localhost:11434

# Optional: Cloud Providers

llm.openai.api_key=${OPENAI_API_KEY}

llm.anthropic.api_key=${ANTHROPIC_API_KEY}

Zero Configuration Needed!

Ollama works out of the box with no API keys required.
